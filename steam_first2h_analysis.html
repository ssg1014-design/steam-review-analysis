🔍 핵심 인사이트

1. 각 게임의 구간별 추천률과 해당 게임의 전체 추천률 간 피어슨 상관계수를 구한 결과, 첫 2시간 추천률은 전체 추천률과 r = 0.83의 강한 상관관계를 보임. 첫 2시간 추천률만으로 게임의 최종 추천률을 69%의 정확도로 예측할 수 있다는 뜻.

2. 2~10시간(R²=0.86), 10~50시간(R²=0.88) 구간의 예측력이 더 높지만, 리뷰 비중 대비 예측력 관점에서 첫 2시간의 효율아 압도적. 전체 리뷰의 13.6%에 불과한 구간이 69%의 예측력을 지니기 때문.

3. 무료 게임의 초기 R²가 0.75로 유료(0.69)보다 높은데, F2P는 진입 장벽이 없어 유저가 더 쉽게 이탈하므로, 첫인상의 결정력이 유료 게임보다 큼

4. 50시간+ 구간의 R²는 0.80으로 2~50시간 구간보다 낮음. 이는 장기 플레이어의 평가가 게임 업데이트·메타 변화·커뮤니티 이슈 등 외부 요인에 영향을 받아 초기 품질과의 상관이 약해지기 때문으로 보임.

방법론

분석 방법

데이터 — Steam Web API에서 수집한 전체 리뷰 154,842,414건 중 리뷰 1만 건 이상인 1,510개 게임을 분석 대상으로 선정.

세그먼트 분류 — 각 리뷰의 playtime_at_review(리뷰 작성 시점의 누적 플레이타임)를 기준으로 <2시간 / 2~10시간 / 10~50시간 / 50시간+.

리뷰 비중 — 게임별로 각 세그먼트에 속하는 리뷰의 비율을 계산하고, 이를 게임 단위로 평균(비가중 평균, 1게임 = 1표).

추천률 산출 — 게임별로 각 세그먼트의 추천률(긍정 리뷰 비율)을 계산하고, 이를 게임 단위로 평균(비가중 평균). 유료·무료 구분은 steam_purchase기준.

상관분석 — 각 게임의 구간별 추천률과 해당 게임의 전체 추천률 간 피어슨 상관계수(r) 및 결정계수(R²)를 산출.

결론

가설 검증 결과

✅ "시장은 게임을 매우 빠르게 판단합니다"

1. 1만 리뷰 이상의 게임 1,517개의 리뷰 데이터 분석 결과, 첫 2시간 이내 리뷰의 추천률만으로 게임 전체 평판의 69~75%를 예측할 수 있음.

2. 이 패턴은 유료·무료를 불문하고 일관되며, 특히 무료 게임에서 예측력이 더 높음(R²=0.75).

3. 전체 리뷰의 13.6%에 불과한 첫 2시간 리뷰가 이 정도의 예측력 → 게임의 최종 평판 방향이 출시 직후 매우 이른 시점에 이미 형성된다는 것을 의미.

4. 사용자의 초기 경험이 게임의 성패를 가를 정도로 매우 중요.
